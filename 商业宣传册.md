### 示例：双均线策略

```python
# 1. 导入策略
from pytdx2.backtest import SMAStrategy, BacktestEngine

# 2. 配置参数
strategy = SMAStrategy({
    'fast_window': 10,  # 快线10日
    'slow_window': 20   # 慢线20日
})

# 3. 运行回测
engine = BacktestEngine(initial_capital=1000000)
result = engine.run_backtest(data, strategy, '000001.SZ')

# 4. 查看结果
print(f"总收益率: {result['total_return']*100:.2f}%")
print(f"夏普比率: {result['sharpe_ratio']:.2f}")
```
