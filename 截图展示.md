## 🚀 快速开始

### 生成图表

```python
from tdx_mcp.backtest.visualization import ChartGenerator
import pandas as pd

# 1. 创建图表生成器
chart = ChartGenerator()

# 2. 加载数据
data = pd.read_csv('stock_data.csv')

# 3. 生成K线图
chart.plot_candlestick(
    data,
    title='平安银行K线图',
    ma_periods=[5, 10, 20],
    volume=True,
    save_path='output.png'
)
```

### 回测可视化

```python
# 1. 运行回测
result = engine.run_backtest(data, strategy, '000001.SZ')

# 2. 生成回测报告图
chart.plot_backtest_result(
    daily_values=result['daily_values'],
    trades=result['trades'],
    title='策略回测结果',
    save_path='backtest.png'
)
```

## 🎨 自定义样式

### 修改配色

```python
# 自定义颜色方案
chart = ChartGenerator(style={
    'up_color': '#FF4136',    # 上涨红色
    'down_color': '#2ECC40',  # 下跌绿色
    'background': '#FFFFFF',  # 背景白色
    'grid': '#EEEEEE'         # 网格浅灰
})
```

### 调整尺寸

```python
# 自定义图表尺寸
chart.plot_candlestick(
    data,
    figsize=(16, 9),  # 16:9宽屏
    dpi=300           # 高清输出
)
```
